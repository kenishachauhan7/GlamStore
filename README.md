# GlamStore 💄

A beauty e-commerce web application built with PHP and MySQL.

## Features
- User Authentication (Login/Register)
- Product Catalog with Categories
- Shopping Cart
- Product Reviews

## Tech Stack
- PHP, MySQL, HTML, CSS, JavaScript
- XAMPP (local development)
- InfinityFree (hosting)
